WeatherSphere

REAL-TIME WEATHER APPLICATION

📋 Project Overview

WeatherSphere is a modern weather application that provides real-time weather information and forecasts for any location worldwide. It features a dynamic UI that changes based on the current weather conditions.

✨ Key Features

  • Real-time weather data from OpenWeatherMap API
  • 7-day weather forecast
  • Location-based weather detection
  • Dynamic backgrounds based on weather (Sunny, Rainy, etc.)
  • Detailed metrics (Humidity, Wind Speed, UV Index)
  • Multi-language support

🛠️ Technologies Used

JavaScript (ES6+) OpenWeather API Tailwind CSS Axios Framer Motion
