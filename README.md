<a name="english"></a>
## 🚀 English

This is a modern, responsive personal portfolio website developed for Celal Arel Temel. It showcases projects, skills, certificates, and professional information.

### ✨ Features
- **Responsive Design:** Optimized for Desktop, Tablet, and Mobile.
- **Modern Theme:** Professional Navy-White light theme based on modern UI standards.
- **Dynamic Components:** Integrated hamburger menu, submenus, and project grids.
- **Gallery Section:** Dedicated page for professional certificates and documents.
- **Project Details:** Detailed sub-pages for individual software projects.

### 🛠️ Technologies Used
- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- JavaScript (DOM Manipulation, Navigation logic)

### 📂 Structure
- `index.html`: Landing page with featured projects.
- `about.html`: Information about goals and background.
- `skills.html`: Technical competencies and soft skills.
- `projects.html`: Full list of software projects.
- `gallery.html`: Certificates and education documents.
- `contact.html`: Contact information and messaging form.
- `projeler/`: Directory for specific project detail pages.
- `css/`: Styling files.
- `js/`: Interactivity scripts.

## 🚀 Türkçe

Bu, Celal Arel Temel için geliştirilmiş modern ve duyarlı (responsive) bir kişisel portföy web sitesidir. Projeleri, yetenekleri, sertifikaları ve profesyonel bilgileri sergiler.

### ✨ Özellikler
- **Duyarlı Tasarım:** Masaüstü, Tablet ve Mobil cihazlar için optimize edilmiştir.
- **Modern Tema:** Modern kullanıcı arayüzü standartlarına dayalı profesyonel Lacivert-Beyaz açık tema.
- **Dinamik Bileşenler:** Entegre hamburger menü, alt menüler ve proje ızgaraları.
- **Galeri Bölümü:** Profesyonel sertifikalar ve belgeler için özel sayfa.
- **Proje Detayları:** Bireysel yazılım projeleri için ayrıntılı alt sayfalar.

### 🛠️ Kullanılan Teknolojiler
- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- JavaScript (DOM Manipülasyonu, Navigasyon mantığı)

### 📂 Yapı
- `index.html`: Öne çıkan projelerin olduğu ana sayfa.
- `about.html`: Hedefler ve geçmiş hakkında bilgiler.
- `skills.html`: Teknik yetkinlikler ve sosyal beceriler.
- `projects.html`: Yazılım projelerinin tam listesi.
- `gallery.html`: Sertifikalar ve eğitim belgeleri.
- `contact.html`: İletişim bilgileri ve mesaj formu.
- `projeler/`: Proje detay sayfaları dizini.
- `css/`: Stil dosyaları.
- `js/`: Etkileşim scriptleri.
